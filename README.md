# K230 扑克牌识别 - 多种精度优化方案对比

本项目提供 K230 开发板扑克牌识别的多种精度优化方案对比程序。

## 硬件配置

- **开发板**：嘉立创庐山派 K230 (CanMV)
- **摄像头**：gc2093
- **输出接口**：UART2 (115200, 8N1)
- **引脚配置**：
  - TX = GPIO11
  - RX = GPIO12

## 程序文件

| 文件 | 说明 |
|------|------|
| `A1.PY` | 基础版 - 原始检测，无任何优化 |
| `A2.PY` | 方案一：时序滤波 - 连续2帧检测到相同目标才输出 |
| `A3.PY` | 方案二：平滑滤波 - 维护标签历史，连续出现多次才输出 |
| `A4.PY` | 方案三：ROI区域 - 只检测画面中心70%区域 |
| `A5.PY` | 方案四：阈值过滤 - 降低置信度阈值+误检过滤 |
| `A6.PY` | 方案六：投票机制 - 5帧窗口投票，统计方法输出 |
| `A7.PY` | 方案七：卡尔曼滤波 - 目标跟踪，平滑预测 |

## 模型文件

**注意**：模型文件较大（3.1MB），请从训练目录获取并自行部署到 SD 卡。

模型路径配置：`/sdcard/mp_deployment_source/3.kmodel`

## 方案对比

| 方案 | 延迟 | 稳定性 | 内存占用 | 推荐场景 |
|------|------|--------|----------|----------|
| A1 基础版 | 无 | 低 | 低 | 实时性优先 |
| A2 时序滤波 | 1-2帧 | 中 | 低 | 一般使用 |
| A3 平滑滤波 | 2-3帧 | 高 | 低 | 减少抖动 |
| A4 ROI区域 | 无 | 中 | 低 | 减少边缘误检 |
| A5 阈值过滤 | 无 | 中 | 低 | 提高召回率 |
| A6 投票机制 | 3-5帧 | 高 | 低 | 统计稳定输出 |
| A7 卡尔曼滤波 | 无 | 最高 | 中 | 精确跟踪 |

## UART2 输出格式

```
label:score,label:score,...
```

**示例**：`Ah:0.95,Kh:0.87,`
